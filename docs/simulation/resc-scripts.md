# RESC Scripts

The package includes:

- `run.resc` for interactive simulation
- `test.resc` for automation or headless use

## `run.resc`

Used when an engineer wants to:

- launch Renode manually
- inspect console behavior
- interact with the CLI

## `test.resc`

Used when automation wants to:

- load the same ELF image
- run without GUI assumptions
- support repeatable CI or Robot execution
